# Dotfiles

My dotfiles for system configuration, requiring:

- `hyprland` as desktop environment, window manager, compositor
- `kitty` as terminal
- `zsh` as shell
- `btop` as system / resources monitoring
- `neovim` as text editor
- `ranger` as file explorer
- `firefox` as browser
- `git` as version control

`hyprland` and `kitty` were already previously installed on my machine

```bash
emerge app-shells/zsh sys-process/btop app-editors/neovim app-misc/ranger www-client/firefox dev-vcs/git
```
